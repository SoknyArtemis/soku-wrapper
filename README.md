# Soku Swarm Syncer

一个在你关则的时候帮你把 Swarm 同步关掉的工具。

如果你用它启动非想天则，关则的时候就能把 Swarm 和 TSK 一招带走，不用再吭哧瘪肚点点点了。

## 🚀 使用方法

把 `Release` 中的 `.exe` 放到 `/th123/` 的**上级目录**，双击运行。

如果显示「Windows 已保护你的电脑」，可以点 `其它信息 -> 仍要运行` 。因为我没钱买证书才提示这个，要是你信不过，可以自行编译源码来用。
